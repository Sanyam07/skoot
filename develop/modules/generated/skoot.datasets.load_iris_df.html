skoot.datasets.load_iris_df¶

skoot.datasets.load_iris_df(include_tgt=True, tgt_name='species', names=None)[source][source]¶

Get the iris dataset.

Loads the iris dataset into a dataframe with the target set as the “species” feature or whatever name is specified in tgt_name.

Parameters:

include_tgt : bool, optional (default=True)

Whether to include the target

tgt_name : str or unicode, optional (default=”species”)

The name of the target feature

names : iterable or None

The column names for the dataframe. If not defined, will default to the feature_names attribute in the sklearn bunch instance.
Returns:

X : pd.DataFrame, shape=(n_samples, n_features)

The loaded iris dataset.
