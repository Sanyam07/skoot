skoot.utils.timed_instance_method¶

skoot.utils.timed_instance_method(attribute_name=None)[source][source]¶

Function timer decorator.

This function is used to decorate instance-based methods in BasePDTransformer classes. In particular, fit and transform. After computing the result of the decorated method, the run time is bound to the instance under the attribute param_name.

Parameters:

attribute_name : str or unicode, optional (default=None)

The name of the attribute under which to save the runtime of the decorated method. This is bound to the instance class. If None, will be set to “<method_name>_time_”.
